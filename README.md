# htd-ha-card

A custom Home Assistant dashboard card for controlling HTD (Home Theater Direct) whole-home audio systems.

## Intent

This card provides a clean, purpose-built UI for HTD zone control directly within the Home Assistant dashboard — going beyond what the generic media player card offers.

Planned features:
- Zone on/off with volume and source selection
- Visual zone layout matching physical room arrangement
- EQ controls (bass, treble, balance) per zone
- Do Not Disturb (DND) toggle per zone
- Party mode / all-zones control
- Designed specifically for the HTD Lync 6/12 and MCA-66 systems
